# Redis Overview

## What is Redis?
Redis (Remote Dictionary Server) is an in-memory data structure store used as a database, cache, and message broker.

## Key Features
- In-memory storage (very fast)
- Key-value store
- Supports data structures: String, List, Set, Hash, Sorted Set
- Persistence (RDB, AOF)
- Replication & High Availability

## Use Cases
- Caching (most common)
- Session storage
- Rate limiting
- Leaderboards (sorted sets)
- Pub/Sub messaging

## Advantages
- Extremely low latency
- High throughput
- Simple to use

## Limitations
- Memory bound
- Not ideal for complex queries
