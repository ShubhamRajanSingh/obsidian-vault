# 🗄️ Database Sharding

#tags: #systemdesign #component #database #scaling

## 📌 What is Sharding?
Splitting data across multiple databases to handle scale.

---

## 🧠 Types
- Range-based
- Hash-based
- Geo-based

---

## ⚖️ Tradeoffs
- Complexity in queries
- Rebalancing difficulty

---

## 🔥 Use Cases
- Large-scale systems (Uber, Facebook)
