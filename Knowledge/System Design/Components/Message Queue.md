# 📬 Message Queue

#tags: #systemdesign #component #queue #async

## 📌 What is a Message Queue?
A system that enables asynchronous communication between services.

---

## 🧠 Why Use MQ?
- Decoupling services
- Handling spikes (buffering)
- Retry & fault tolerance

---

## ⚙️ Tools
- Kafka
- RabbitMQ
- AWS SQS

---

## ⚖️ Tradeoffs
- Eventual consistency
- Message ordering challenges

---

## 🔥 Use Cases
- Notification systems
- Logging pipelines
- Background jobs
