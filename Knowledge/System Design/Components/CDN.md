# 🌍 CDN (Content Delivery Network)

#tags: #systemdesign #component #cdn #scaling

## 📌 What is CDN?
Distributed servers that deliver content closer to users.

---

## 🧠 Why CDN?
- Reduce latency
- Improve availability
- Reduce origin server load

---

## ⚙️ How it Works
- Edge servers cache content
- Requests routed to nearest location

---

## ⚖️ Tradeoffs
- Cache invalidation
- Cost vs performance

---

## 🔥 Use Cases
- Video streaming
- Static assets (images, JS, CSS)
